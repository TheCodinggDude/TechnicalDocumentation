Introduction

C is a programming language developed at AT & T's Bell laboratory in USA in 1972. It was designed and written by Dennis Ritchie. C quickly became popular because it is simple, reliable and easy to use. In an industry where new technologies come and go every day, a language that has survived for 3 years has to be really good.

Why you should learn C?

  • It is very difficult to learn C++ or Java directly.This is because while learning these languages you have to learn about OOPS concepts. Learning such complicated concepts while you are not even comfortable with basic concepts is like pulling the cart before the horse. Hence one should learn all language elements very thoroughly using C before migrating to C++/Java/Python.
  • Major parts of Windows operating system such as Windows, UNIX, Linux is still written in C. Hence when it comes to performance, nothing beats C.
  • Electronic devices such as Microwave Oven, Washing Machine and digital cameras use a program embedded in their devices. They not only have to run fast but also have to work in limited amount of memory. They are also written in C language.
  • Many popular gaming frameworks are also built using C language.

Variables

A variable is nothing but a name given to a storage area that our programs can manipulate. Each variable in C has a specific type, which determines the size and the variable's memory, the range of values that can be stored within that memory and the set of operations that can be applied to the variable.

Keywords

Keywords are the words whose meaning has already been explained to the C compiler. The keywords canot be used as variable name because if we do we are trying to assign a new meaning to the keyword,which is not allowed by the computer.

There are 32 keywords available in C. Following is the list of keywords in C

unsigned void volatile while

Identifiers

Identifier refers to name given to entities such as variables, functions, structures etc. Identifiers must be unique. They are created to give a unique name to an entity to identify it during the execution of the program. For example:

int money;
double accountBalance;

Rules for naming Identifiers

  • A valid identifier can have letters (both uppercase and lowercase letters), digits and underscores.
  • The first letter of an identifier should be either a letter or an underscore.
  • You cannot use keywords like int, break etc. as identifiers.
  • There is no rule on how long an identifier can be. However, you may run into problems in some compilers if the identifier is longer than 31 characters.

Operators

An operator is a symbol that operates on one or more values or variables. For example: + is an operator to perform addition.

C has a wide range of operators to perform various operations.

  1. Arithmetic Operators

    Operator Meaning of Operator
    + addition or unary plus
    - subtraction or unary minus
    * multiplication
    / division
    % remainder after division (modulo division)

  2. Increment and Decrement Operators

    C programming has two operators increment (++) and decrement (--) to change the value of an operand (constant or variable) by 1.

    Increment (++) increases the value by 1 whereas decrement (--) decreases the value by 1. These two operators are unary operators, meaning they only operate on a single operand.

  3. Assignment Operators

    Operator Example Same as
    = a = b a = b
    += a += b a = a+b
    -= a -= b a = a-b
    *= a *= b a = a*b
    /= a /= b a = a/b
    %= a %= b a = a%b

  4. Relational Operators

    A relational operator checks the relationship between two operands. If the relation is true, it returns 1; if the relation is false, it returns value 0.

    Relational operators are used in decision making and loops.

    Operator Meaning of Operator
    Example
    == Equal to 5 == 3 is evaluated to 0
    > Greater than 5 > 3 is evaluated to 1
    < Less than 5 < 3 is evaluated to 0
    != Not equal to 5 != 3 is evaluated to 1
    >= Greater than or equal to 5 >= 3 is evaluated to 1
    <= Less than or equal to 5 <= 3 is evaluated to 0

  5. Logical Operators

    An expression containing logical operator returns either 0 or 1 depending upon whether expression results true or false. Logical operators are commonly used in decision making in C programming.

    Operator
    Meaning
    Example
    && Logical AND. True only if all operands are true If c = 7 and d = 3 then, expression ((c==7) && (d>7)) equals to 0.
    || Logical OR. True only if either one operand is true If c = 7 and d = 3 then, expression ((c==7) || (d>7)) equals to 1.
    ! Logical NOT. True only if the operand is 0 If c = 7 then, expression !(c==7) equals to 0.

  6. Bitwise operators

    During computation, mathematical operations like: addition, subtraction, multiplication, division, etc are converted to bit-level which makes processing faster and saves power.

    Bitwise operators are used in C programming to perform bit-level operations.

    Operators Meaning of operators
    & Bitwise AND
    | Bitwise OR
    ^ Bitwise exclusive OR
    ~ Bitwise complement
    << Shift left
    >> Shift right

Rules applicable to all C programs

  1. Each instruction in a C program is written as a seperate statement. Therefore a complete C program would comprise of a series of statements.
  2. The statements in the program must appear in the same order in which we wish them to be executed.
  3. All statements are entered in lowercase letters.
  4. Every C statement must end with a ;. Therefore ; acts as a statement terminator.
